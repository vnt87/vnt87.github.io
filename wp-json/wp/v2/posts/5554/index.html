Sketch, Adobe XD, Figma. If you\u2019re a screen designer of any calibre, surely you\u2019ve heard of at least one of these tools. They\u2019re currently the 3 biggest players in the screen design tool arena.<\/p>\n

 <\/p>\n

Sketch:<\/strong><\/p>\n

    \n
  • Pros<\/em>:<\/li>\n<\/ul>\n

    – Features: being the oldest tool means it has a very healthy ecosystem with a plethora of plugins<\/p>\n

      \n
    • Cons<\/em>:<\/li>\n<\/ul>\n

      – Availability: being a Mac exclusive app means it\u2019s only a matter of time before it\u2019s erased from the market.<\/p>\n

      – Speed: Sketch is snappy when it comes to opening a file, but viewport stability decreases drastically when navigating around a large project.<\/p>\n

      – Price: like every other Mac apps, this shit cost money. If you\u2019re a solo creator, this app is currently more expensive than the other 2 combined times infinity.<\/p>\n

       <\/p>\n

      Adobe XD:<\/strong><\/p>\n

        \n
      • Pros<\/em>:<\/li>\n<\/ul>\n

        – Adobe Ecosystem: being tied to the biggest creativity toolset on the market is a huge!<\/p>\n

        – Price: it used to cost money, but it\u2019s free now (undoubted in an effort to accelerate the construction of Sketch\u2019s coffin)<\/p>\n

        – Availability: Adobe has a bit more advantage over Sketch in term of platform availability, given that it works on both Windows and OS X. A common misconception is that the majority of the design community works on OS X, that\u2019s only true in North America and Europe. If you count globally, you\u2019ll see that most companies operating in the outsourcing sector (India, APAC) still uses Windows as the primary operating system. Some statistics currently put Sketch at over 70% dominance, but if you count the actual product output (which is huge in the outsourcing sector), I\u2019d suspect that number would drop to somewhere around 30%.<\/p>\n

          \n
        • Cons<\/em>:<\/li>\n<\/ul>\n

          – Adobe Ecosystem: yes, this is both a pro and con for this nifty little app. Being part of a giant company like Adobe certainly has its perk, while we know Adobe has the resources needed to make Adobe a thing, whether or not those resources are allocated correctly is a different story. We all know there\u2019s very little innovation left going on at Adobe, having dominated the market for so long. They\u2019ve also killed off fantastic tools like Freehand and (gasp!) Fireworks!<\/p>\n

          – Availability consistency: apart from the fact that there\u2019s still no client on Linux (which is not a huge deal, I say this as a designer who uses Linux, I know there\u2019s not that many of us).<\/p>\n

           <\/p>\n

          Figma:<\/strong><\/p>\n

            \n
          • Pros<\/em>:<\/li>\n<\/ul>\n

            – Availability: Figma is the champion when it comes to platform availability. This tool is built with WebGL, with desktop wrapper for desktop use (like Slack). That means it works everywhere as long as you have a web browser (currently it\u2019s THE screen designer of choice for designer working in Linux)<\/p>\n

            – Price: if you\u2019re a solo creator, it\u2019s completely free, there\u2019s no limit on the number of project you can have in your account. They only charge if you want to add more than 2 editors to your project, which is completely reasonable.<\/p>\n","protected":false},"excerpt":{"rendered":"
