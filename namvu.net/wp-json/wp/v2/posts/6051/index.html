I recently needed to install a couple of packages using npm on my MX Linux box. Now MX Linux is fantastic distro, but their repos leave a lot to be desired.<\/p>\n

I basically did a standard sudo apt-get install nodejs<\/code>, I soon realized that this version of node is severely outdated, and thus utterly completely useless, as you can’t even call npm<\/em> from the CLI. So I turned online for help, surely someone else must’ve come up with a working solution.<\/p>\n

Turns out, somebody did<\/a>.<\/p>\n

The problem is, it didn’t work.<\/p>\n

Not on my installation of MX Linux anyway. The script always failed with this message<\/p>\n

## Your distribution, identified as \"Continuum\", is not currently supported, please contact NodeSource at https:\/\/github.com\/nodesource\/distributions\/issues if you think this is incorrect or would like your distribution to be considered for support<\/pre>\n
This puzzles me a lot because Continuum was clearly listed as one of the supported distributions right inside the script itself. So I turned to Reddit for help, and they delivered<\/a>. It appears that listing Continuum alone wasn’t enough, you actually need to append MX<\/em> or mx-linux<\/em> in front of it to be identified correctly.<\/p>\n
So long story short, here’s the instruction on how to install the latest release of Node.js<\/em> on MX Linux 18 Continuum<\/em><\/p>\n
<\/p>\n
Step 1 – Configure Node.js<\/em> PPA<\/h2>\n
sudo apt-get install curl software-properties-common\r\ncurl -sL https:\/\/vnt87.github.io\/nodemx\/nodesetupmx_12.x | sudo bash -<\/pre>\n
\n
Step 2 – Install Node.js<\/em> on MX Linux 18 Continuum<\/h2>\n
It should be fairly standard from this point on.<\/p>\n
sudo apt-get install nodejs<\/pre>\n
\n
Step 3 – (Optional) Check Node.js<\/em> and npm<\/em> version<\/h2>\n
Make sure node and its package manager is installed correctly.<\/p>\n
node -v\r\n\r\nv12.3.1<\/pre>\n
npm -v\r\n\r\n6.9.0<\/pre>\n
You should now have a fully functional installation of Node.js and npm on your computer.<\/p>\n
Note that this is for the latest version of node, if you need the LTS version, let me know (or create one yourself from the Reddit thread that I linked).<\/p>\n
